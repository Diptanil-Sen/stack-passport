## вңҰ What is Stack Passport?

Stack Passport is a **holographic developer identity card generator**. You fill in your tech stack, years of experience, and role вҖ” it generates a cinematic, spy-file aesthetic passport card that you can download as a PNG and share everywhere.

Think of it as your developer flex. Your LinkedIn profile picture for people who actually code.

---

## вҡЎ Features

- **5 Card Themes** вҖ” Cyber Blue, Hacker Green, Blood Red, Midnight Purple, Golden Hour
- **8 Passport Types** вҖ” Full Stack, Frontend, Backend, DevOps, Mobile, ML, Security, Game Dev
- **Auto Level System** вҖ” RECRUIT вҶ’ ENGINEER вҶ’ SENIOR вҶ’ PRINCIPAL вҶ’ LEGEND based on your stack
- **Visa Stamp Grid** вҖ” Every technology gets its own stamp with icon and years of experience
- **MRZ Zone** вҖ” Machine-readable zone at the bottom, like a real passport
- **Holographic Effects** вҖ” Scan line, holo shimmer, 3D tilt on hover
- **Avatar Upload** вҖ” Upload your photo or use auto-generated initials
- **Download PNG** вҖ” High quality 2x export with watermark
- **Shareable Links** вҖ” Every passport gets a unique URL, shortened via TinyURL
- **Share Buttons** вҖ” Twitter, LinkedIn, WhatsApp, Instagram
- **Confetti** вҖ” Because you deserve it
- **Live Preview** вҖ” See your card update as you type
- **Fully Responsive** вҖ” Works on mobile, tablet, desktop

## рҹҡҖ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/Diptanil-Sen/stack-passport.git

# Navigate into the project
cd stack-passport

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) and start building your passport.

## рҹҸҶ Level System

Levels are auto-calculated based on your tech count and years of experience:

| Level | Score | Color |
|-------|-------|-------|
| RECRUIT | < 12 | `#64748b` |
| ENGINEER | 12вҖ“24 | `#00ff88` |
| SENIOR | 25вҖ“39 | `#00d4ff` |
| PRINCIPAL | 40вҖ“59 | `#bf00ff` |
| LEGEND | 60+ | `#fbbf24` |

> **Score** = `(number of technologies Г— 2) + (years of experience Г— 3)`

## рҹ”— How Sharing Works

Every generated passport is encoded directly into the URL as a compressed base64 string вҖ” **no backend, no database, no login required**. When someone opens the link, the passport is decoded and rendered client-side instantly.

When you click **Copy Link**, the URL is automatically shortened via [TinyURL](https://tinyurl.com) before being copied to your clipboard.

## рҹ“Ұ Deployment

This project is deployed on Vercel with client-side routing handled via `vercel.json`:

```json
{
  "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
}
```

To deploy your own instance:

1. Fork this repo
2. Go to [vercel.com](https://vercel.com) вҶ’ Import project
3. Select your fork вҖ” leave all settings default
4. Click Deploy
